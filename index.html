☁️ Gist sync
Sync your data across devices via a private GitHub Gist. Each person uses their own Gist + token — no shared infrastructure needed.
The hash at the end of your Gist URL: gist.github.com/yourname/this-part
github.com/settings/tokens → Generate new token (classic) → gist scope only
How to set up:
1. Go to gist.github.com → New gist → filename: pokehub.json → content: {} → Create secret gist
2. Copy the Gist ID from the URL and paste above
3. Go to github.com/settings/tokens → Generate new token → check gist only → copy and paste above
4. Hit "Sync now" — your data will push to the Gist and pull on every other device
